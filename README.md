# Recipe Helper

An open-source Minecraft Forge mod for dynamically managing and generating crafting recipes at runtime.

The mod provides a graphical interface and command system to create, edit, reload, and export recipes using standard Minecraft JSON formats. It is designed for modpack developers and advanced users who need flexible, runtime recipe control without restarting the game.

This project is actively developed and released under an open-source license.

## Features
- Runtime recipe creation and modification via GUI
- Command-based control with permission level 2 (OP)
- Supports exporting recipes to KubeJS and CraftTweaker
- Compatible with multiple popular mods
- Extensible recipe processor system based on JSON configuration
